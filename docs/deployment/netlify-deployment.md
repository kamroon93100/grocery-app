\# Frontend Deployment



\## Platform



Netlify



\## Build Command



```bash

flutter build web

```



\## Publish Folder



```

build/web

```



\## Environment



\- Production API URL

\- HTTPS Enabled

\- Responsive Web Build



\## Deployment Flow



```text

GitHub

&#x20;   ↓

Netlify

&#x20;   ↓

Build Flutter Web

&#x20;   ↓

Deploy

```
