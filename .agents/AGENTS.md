# AGENTS.md

## Mission

Build a world-class grocery delivery frontend inspired by the best UX patterns found in modern applications.

Reference products include:

- Swiggy Instamart
- Blinkit
- Zepto
- Apple
- Linear
- Notion
- Airbnb
- Spotify

The objective is NOT to copy code or assets.

Instead:

- Study layouts
- Study spacing
- Study typography
- Study animations
- Study interaction patterns
- Study user flows
- Recreate similar experiences using original code and assets.

---

## Design Research Workflow

For every screen:

1. Analyze visual hierarchy.
2. Identify reusable UI patterns.
3. Measure spacing and alignment.
4. Study animation timing and easing.
5. Note responsive behavior.
6. Build an original component that achieves the same usability and feel.

---

## Component Requirements

Each component must:

- Be built from scratch.
- Use original code.
- Use original SVGs or open-source icon libraries.
- Be reusable and configurable.
- Follow the project's design system.
- Match performance and responsiveness.

---

## Inspiration Sources

Study:

- Navigation bars
- Hero banners
- Product cards
- Search bars
- Carousels
- Bottom sheets
- Floating buttons
- Skeleton loaders
- Toasts
- Checkout flows
- Cart interactions
- Micro-animations

Extract the design principles, not proprietary assets.

---

## Animation Rules

Analyze:

- Duration
- Delay
- Spring behavior
- Hover effects
- Press states
- Loading transitions

Implement your own versions using Framer Motion.

---

## UX Goals

Aim for:

- 120 FPS interactions
- Smooth scrolling
- Fast perceived performance
- Minimal visual clutter
- Clear information hierarchy
- Consistent spacing
- Accessible interactions

---

## Forbidden

Never:

- Copy source code.
- Copy proprietary SVGs, icons, logos, or illustrations.
- Copy copyrighted images.
- Clone branded visual identities exactly.
- Replicate trademarked assets.

Instead, create original implementations inspired by common industry best practices.

---

## Success Criteria

Every new component should feel premium, polished, responsive, and performant while remaining an original implementation.
