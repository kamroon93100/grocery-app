- ✅ OTP Authentication
- ✅ Product Categories
- ✅ Product Search
- ✅ Product Details
- ✅ Shopping Cart
- ✅ Checkout Flow
- ✅ Offers

- ✅ Responsive Web
- ✅ Android Support
- ✅ Production Backend
