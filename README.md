\# Kohli Store App



A production-style grocery delivery app built with Flutter.



\## Features



\- Flutter Web and Android support

\- Live product catalog

\- Category browsing

\- Product search

\- Product detail pages

\- Cart and checkout flow

\- Lottie animations

\- Render backend integration

\- Netlify deployment



\## Tech Stack



\- Flutter

\- Dart

\- Provider

\- REST API

\- Netlify

\- Render
